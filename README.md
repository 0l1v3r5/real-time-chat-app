
## Folder Structure

After creation, your project should look like this:

```
my-app/
  README.md
  node_modules/
  app.js
  .gitignore
  package.json
  package-lock.json
  views/
    index.ejs
```

## Installing a Dependency

### `npm install`

Launch this command to install all dependencies

##Running

### `node app.js`

Launches the application in the interactive watch mode.<br>
